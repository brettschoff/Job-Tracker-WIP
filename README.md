
# BS Job Hunter

BS Job Hunter is a utility designed for those seeking employment. BS Job Hunter allows you to record your job applications so you can stay organized and on top of your job search, and keep an organized set of notes all in one place for those late callbacks youâ€™ve already forgotten about. 

## Features

1. User-Authentication using jsonwebtoken and bcrypt

2. Create and manage job applications

3. Add personal notes to individual applications and sort them based on priority
